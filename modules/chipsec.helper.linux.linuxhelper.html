devmem_available()[source]¶

Check if /dev/mem is usable.

In case the driver is not loaded, we might be able to perform the requested operation via /dev/mem. Returns True if /dev/mem is accessible.

devmsr_available()[source]¶

Check if /dev/cpu/CPUNUM/msr is usable.

In case the driver is not loaded, we might be able to perform the requested operation via /dev/cpu/CPUNUM/msr. This requires loading the (more standard) msr driver. Returns True if /dev/cpu/CPUNUM/msr is accessible.
